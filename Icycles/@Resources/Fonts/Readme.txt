========================================
MANUAL INSTALLATION REQUIRED
========================================

Icycles uses JetBrains Mono for its minimalist, technical aesthetic.
Due to Windows security restrictions, fonts cannot be installed automatically.

INSTALLATION STEPS:
1. Download JetBrains Mono from: https://www.jetbrains.com/lp/mono/
2. Extract the downloaded ZIP file
3. Open the "fonts/ttf" folder
4. Select all .ttf files (Ctrl+A)
5. Right-click and select "Install for all users"
6. Restart Rainmeter after installation

REQUIRED WEIGHTS:
- JetBrains Mono Regular (400)
- JetBrains Mono Medium (500)
- JetBrains Mono Bold (700)
- JetBrains Mono ExtraBold (800)

VERIFICATION:
After installation, Icycles will automatically detect and use the font.
If the font is not installed, the system will fall back to Consolas.
